# curse-web

Marketing and documentation website for **curse** — an all-in-one Discord moderation bot
(verification, moderation, information, economy, fun).

Built with **Next.js (App Router) + TypeScript + Tailwind CSS v4**, using **Bun**.

Sleek dark theme: near-black base (`#09090B`) with a single electric-violet accent (`#8B7CF6`).

## Development

```bash
bun install
bun run dev      # http://localhost:3000
bun run build
```

> Site is scaffolded per the implementation plan; CTA invite link is configured via
> `NEXT_PUBLIC_INVITE_URL` (defaults to `#`).
